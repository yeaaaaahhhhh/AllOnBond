## 🎯 프로젝트 개요

**All On Bond**는 채권 투자자를 위한 수익률 계산 및 비교 도구입니다.

### 핵심 기능
1. **계산기**: 채권 정보 입력 → YTM, 예금환산수익률, 현금흐름 계산
2. **채권 비교**: 실시간 채권 정보 크롤링 → 수익률 비교 테이블
3. **계산 설명**: 수식 및 계산 과정 상세 설명

### 기술적 특징
- GitHub Pages 정적 호스팅 (무료)
- 클라이언트 사이드 계산 (서버리스)
- GitHub Actions 자동 데이터 수집
- 원화/달러 이중 지원
- 세전/세후 수익률 계산
